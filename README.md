# Study / Productivity Planner

## 프로젝트 소개

이 프로젝트는 사용자가 일정을 체계적으로 관리하고, 하루하루의 목표 달성 과정을 시각적으로 확인할 수 있도록 만든 **생산성 관리 웹 애플리케이션**입니다.  
단순히 할 일을 적는 데서 끝나는 것이 아니라, **Monthly / Weekly / Daily / Process** 화면을 통해 계획부터 수행, 그리고 달성률 확인까지 한 번에 할 수 있도록 구성했습니다.

사용자는 다양한 화면에서 자신의 일정을 확인하고, 계획한 일을 완료 처리하며, 현재 얼마나 목표를 달성했는지 직관적으로 확인할 수 있습니다.  
이를 통해 일정 관리의 편의성과 함께 꾸준히 실천할 수 있는 동기부여를 제공하는 것을 목표로 했습니다.

This project is a **productivity management web application** designed to help users organize their schedules systematically and visually track their daily progress.  
Rather than simply recording tasks, the application provides **Monthly / Weekly / Daily / Process** pages so that users can experience the full flow of planning, completing tasks, and checking achievement rates in one service.

Users can review their schedules in multiple views, mark planned tasks as completed, and intuitively understand how much of their goals they have achieved.  
Through this process, the project aims to provide both convenience in schedule management and motivation for consistent action.

---

## 주요 기능 | Main Features

### 1. Monthly

**Monthly 페이지**는 한 달 전체 일정을 달력 형태로 확인할 수 있는 화면입니다.  
사용자는 날짜별 일정을 한눈에 파악할 수 있어 장기적인 계획을 세우기에 적합합니다.

The **Monthly page** provides a calendar view of the entire month.  
It allows users to check schedules by date at a glance, making it suitable for long-term planning.

#### 특징 | Features
- 월간 달력 형태로 일정 확인 가능
- 날짜별 계획을 한눈에 볼 수 있음
- 장기적인 일정 관리에 적합

- View schedules in a monthly calendar format
- Check plans by date at a glance
- Suitable for long-term schedule management

---

### 2. Weekly

**Weekly 페이지**는 한 주 동안의 계획을 보다 구체적으로 확인할 수 있는 화면입니다.  
사용자는 주간 단위로 일정을 정리하면서, 단기 목표를 효율적으로 관리할 수 있습니다.

The **Weekly page** helps users review their plans for the current week in more detail.  
This view allows users to organize short-term goals and manage weekly schedules more effectively.

#### 특징 | Features
- 주간 단위 일정 확인 가능
- 짧은 기간의 목표를 정리하기 쉬움
- 주별 계획 흐름을 파악하는 데 유용함

- View schedules on a weekly basis
- Helpful for organizing short-term goals
- Useful for understanding weekly planning flow

---

### 3. Daily

**Daily 페이지**는 하루 단위의 할 일을 상세하게 관리할 수 있는 화면입니다.  
사용자는 새로운 할 일을 계획으로 추가할 수 있으며, 카테고리와 시간 등을 입력하여 일정에 반영할 수 있습니다.

또한 계획한 할 일을 완료하면, 해당 작업은 **Planned Tasks**에서 **Completed Tasks**로 이동하여 완료 여부를 명확하게 확인할 수 있습니다.  
이 과정을 통해 사용자는 오늘 계획한 일 중 어떤 일을 끝냈는지 쉽게 파악할 수 있습니다.

특히 Daily 화면에서는  
**계획한 전체 할 일 중 몇 개를 완료했는지**를 확인할 수 있어,  
하루 목표 달성 정도를 직관적으로 볼 수 있습니다.

The **Daily page** is designed for detailed day-to-day task management.  
Users can add new planned tasks by entering categories and time information.

When a planned task is marked as completed, it moves from **Planned Tasks** to **Completed Tasks**, allowing users to clearly check which tasks have been finished.  
This makes it easy to understand what has been completed from the tasks planned for the day.

In particular, the Daily page also shows  
**how many tasks have been completed out of the total planned tasks**,  
so users can intuitively see their daily achievement level.

#### 특징 | Features
- 하루 할 일 계획 추가 가능
- 카테고리 및 시간 입력 가능
- 완료 버튼을 누르면 완료된 작업 목록으로 이동
- 계획한 작업 수 대비 완료한 작업 수 확인 가능
- 하루의 달성 현황을 직관적으로 파악 가능

- Add daily planned tasks
- Enter category and time information
- Move completed tasks into the completed task list
- Check the number of completed tasks out of total planned tasks
- Easily understand daily progress

---

### 4. Process

**Process 페이지**는 사용자의 전체 진행 상황과 달성률을 시각적으로 보여주는 화면입니다.  
이 페이지에서는 단순히 숫자로만 진행도를 보여주는 것이 아니라,  
**달성률에 따라 토끼 캐릭터 이미지가 달라지는 방식**을 적용해 사용자에게 재미와 동기부여를 제공합니다.

즉, 사용자가 목표를 더 많이 달성할수록 토끼의 상태나 모습이 변화하여,  
진행 상황을 더욱 직관적이고 흥미롭게 확인할 수 있습니다.

The **Process page** visually presents the user’s overall progress and achievement rate.  
Instead of showing progress only as numbers, this page uses  
**different rabbit character images depending on the achievement rate**  
to provide both fun and motivation.

As the user achieves more goals, the rabbit’s appearance changes,  
making the progress easier to understand and more engaging.

#### 특징 | Features
- 전체 진행도 및 달성률 확인 가능
- 달성률에 따라 토끼 이미지가 달라짐
- 사용자의 성취도를 시각적으로 표현
- 지속적인 참여와 동기부여에 도움

- Check overall progress and achievement rate
- Rabbit image changes based on progress level
- Visual representation of user achievement
- Encourages continuous participation and motivation

---

## 프로젝트 목표 | Project Goal

이 프로젝트의 목표는 단순한 일정 관리 서비스를 만드는 것이 아니라,  
사용자가 **계획 → 수행 → 완료 확인 → 달성률 확인**의 흐름을 자연스럽게 경험할 수 있도록 하는 것입니다.

특히 다음과 같은 점에 중점을 두었습니다.

- 월간, 주간, 일간 단위로 일정을 다양한 관점에서 관리할 수 있도록 구성
- 계획한 일을 완료 처리하면서 성취감을 느낄 수 있도록 설계
- 달성률을 시각적으로 제공하여 꾸준한 실천을 유도
- 생산성과 자기관리 능력을 동시에 높일 수 있는 서비스 구현

The goal of this project is not simply to create a schedule management service,  
but to help users naturally experience the flow of  
**planning → doing → checking completion → reviewing achievement rate**.

We especially focused on the following points:

- Providing multiple perspectives for schedule management through monthly, weekly, and daily views
- Designing the service so users can feel a sense of accomplishment by marking tasks as completed
- Encouraging consistent action through visualized achievement rates
- Building a service that improves both productivity and self-management

---

## 기대 효과 | Expected Effect

이 웹 애플리케이션을 통해 사용자는 단순히 해야 할 일을 기록하는 것을 넘어서,  
자신의 일정과 수행 과정을 체계적으로 관리할 수 있습니다.

또한 Daily 페이지에서 완료한 작업 수를 바로 확인하고,  
Process 페이지에서 달성률 변화와 토끼 캐릭터 변화를 확인하면서  
보다 재미있고 지속적으로 목표를 실천할 수 있습니다.

즉, 본 프로젝트는  
**일정 관리의 효율성**과 **사용자 동기부여**를 함께 고려한 서비스입니다.

Through this web application, users can go beyond simply writing down tasks  
and systematically manage both their schedules and execution process.

By checking the number of completed tasks on the Daily page  
and observing changes in both achievement rate and rabbit character on the Process page,  
users can practice their goals in a more engaging and consistent way.

In other words, this project is a service that considers both  
**efficiency in schedule management** and **user motivation**.

---

## 화면 구성 | Pages

- **Monthly**: 한 달 단위 일정 확인 / Monthly calendar-based schedule management
- **Weekly**: 일주일 단위 계획 관리 / Weekly planning and short-term task management
- **Daily**: 할 일 추가, 완료 처리, 완료된 task 확인, 전체 계획 대비 완료 개수 확인 / Add tasks, mark them completed, check completed tasks, and view completed count out of total planned tasks
- **Process**: 달성률 확인 및 달성률에 따라 달라지는 토끼 이미지 제공 / Check achievement rate and view rabbit images that change according to progress

## 마무리 | Conclusion

본 프로젝트는 일정 관리와 진행도 확인을 결합하여,  
사용자가 스스로 계획을 세우고 실천하며 성취도를 확인할 수 있도록 설계한 웹 애플리케이션입니다.  
특히 Monthly, Weekly, Daily, Process의 네 가지 화면을 통해  
사용자가 자신의 목표를 다양한 관점에서 관리할 수 있도록 구성했습니다.

The project combines schedule management with progress tracking  
to help users make plans, carry them out, and check their achievements independently.  
In particular, the four pages — Monthly, Weekly, Daily, and Process —  
allow users to manage their goals from multiple perspectives.
